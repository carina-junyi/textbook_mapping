# 國小影片教材對應工具

  本專案使用 AI 自動比對教學影片與各出版社教科書章節的對應關係，協助教師快速找到適合搭配各版本教材的影片資源。

  ## 支援出版社

  - 康軒
  - 南一
  - 翰林

  ## 功能

  - 將出版社 PDF 教科書轉換為圖檔供 AI 辨識
  - 產出「影片 → 出版社章節」對應表（影片對應出版社_結果.xlsx）
  - 產出「出版社章節 → 影片」對應表（出版社對應影片_結果.xlsx）
  - 自動評估對應符合程度（1～5分），低分項目以紅底標註並附建議

  ## 資料夾結構

  .
  ├── 出版社/              # 存放各出版社 PDF（不含於版本控制）
  ├── video_list.txt       # 影片清單
  ├── 影片對應出版社.csv   # 範本
  ├── 出版社對應影片.csv   # 範本
  └── .claude/
      └── CLAUDE.md        # 專案執行指引

  ## 使用方式

  1. 將各出版社 PDF 放入 `出版社/` 資料夾
  2. 確認 `video_list.txt` 與兩份 CSV 範本存在
  3. 執行 Claude Code，依照 CLAUDE.md 的流程操作

  ## 注意事項

  - PDF 與圖檔體積較大，不納入版本控制
  - 輸出的 xlsx 結果檔亦不納入版本控制
